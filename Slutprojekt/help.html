Help

This page provides help for this website.

Some simple questions and answers are provided here.

Does this page have trackers/ads?

Simply put, no. I don't use any trackers or ads on this page.

I don't want to collect any data from anyone. I just want to share knowledge.

I respect my privacy and I want to respect yours too.

Why does the colors on this page look weird?

You might have a color filter on your device that changes the colors.

Try disabling these filters to see the page as intended.

If it still looks weird, then it might be that you dislike the colors that I used.

I'm sorry if that's the case. But I won't change it.

If you want to see where I got my colors from, you can check this github repository:

Catppuccin

Why do stuff disappear sometimes from the page?

You might have an extension that suddenly removes stuff from the page.

Try disabling these extensions to see if the problem persists.
